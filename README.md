# hcifootprint-demo

Demos for [HCIFootprint](https://github.com/footprintjs/hcifootprint) — each demo is a
self-contained folder with its own `package.json` and tests.

| Demo | Story |
|---|---|
| [`dress-shop/`](dress-shop/) | A vanilla e-commerce app becomes agent-operable in three commits — the diff is the demo (commit 1: the app alone · commit 2: the agent layer, zero app lines changed · commit 3: an agentfootprint assistant with pause/resume order confirmation). |
